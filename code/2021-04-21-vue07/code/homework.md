### 任务

- 实现完整版 todo-list (10 分)
  - 可以通过输入框输入一个 todo（1 分）
  - 可以删除一个 todo（1 分）
  - 可以完成一个 todo （1 分）
    - 点击完成按钮后，当前 todo 的样式画个横线
  - all 按钮，显示所有 todo（3 分）
  - active 按钮，显示所有处于 active 状态下的所有 todo（2 分）
  - completed 按钮，显示所有处于 completed 状态下的所有 todo（2 分）
- 技术要求
  - 使用 vue3 的 composition api 方式编写逻辑
  - 使用 vuex
  - 使用 element3
    - 使用 button 组件即可
